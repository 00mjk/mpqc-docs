tries to translate a library basis set label to the corresponding default value for the DF basis

Parameters
obs_nameorbital basis set name; to be useful must be a canonical library name
incXoptional parameter to raise the cardinal number of the density fitting basis. This may be useful if using density-fitting slightly outside their intended area of application.
force_augoptional parameter similar to incX to force inclusion of diffuse components of the density fitting basis
Returns
canonical library name of the default density-fitting basis set; if not able to suggest the default basis, returns an empty string
