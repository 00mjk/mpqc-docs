Detailed Description

DistArray4_MemoryGrp handles transformed integrals held in memory by MemoryGrp.

The ordering of integrals in MemoryGrp buffers is not specified to avoid having to reorder integrals Each pair block has size of num_te_types*nbasis1*nbasis2

Stores an ij pair block of integrals.

It is assumed to be stored according to storage(). It does not have to be "local", i.e. is_local(i,j) can be false, but:

  • nonlocal writes may require additional copies and/or communication.
  • and currently fail for some implementations :-( Locking is performed, hence should be thread safe
