Detailed Description

This specialization of AccumH computes the contribution to the energy and one body Hamiltonian from a solvent using a polarizable continuum model.

WARNING: The BEMSolventH class is not thoroughly tested. It should only be used by developers wishing to test or fix it.

Should be called after we are finished with this AccumH.

The reference to current Wavefunction object will be removed, and accum cannot be called until another init call is made.
