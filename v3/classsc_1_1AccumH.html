Detailed Description

AccumH computes additions to the one body Hamiltonian.

Specializations of AccumH can be given to derivatives of the SCF class, and there they will be used to modify the one body Hamiltonian.

Constructor & Destructor Documentation

◆ AccumH()

sc::AccumH::AccumH ( const Ref< KeyVal > &  )

The KeyVal constructor.

wavefunction

This gives a Wavefunction object. Sometimes additional contributions to the Hamiltonian depend on the current Wavefunction, and, in these cases, the contribution must be obtained iteratively. It is usually not necessary to specify this, since it is given in the init call below.
