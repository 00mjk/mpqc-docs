◆ MPQC_CLASS_EXPORT

#define MPQC_CLASS_EXPORT (   ...)
Value:
MPQC_CLASS_EXPORT_KEY(__VA_ARGS__) \
MPQC_FORCELINK_KEYVAL_CTOR(__VA_ARGS__)

For a class T derived from DescribedClass MPQC_CLASS_EXPORT(T), creates a Boost.Serialization GUID for the class, registers the KeyVal ctor of T (see MPQC_CLASS_EXPORT_KEY(T)), and allows to force linking of T into the executable by defining a ForceLink<T> object in it (see MPQC_FORCELINK_KEYVAL_CTOR(T)). Place MPQC_CLASS_EXPORT(T) to a source file in global scope.

Note
To avoid slowing down compilation by coupling the instantiation of multiple classes, use only one MPQC_CLASS_EXPORT(T) per .cpp file

Streamlines the definition of ForceLink<T>::create methods.

For every object of ForceLink<T> type defined in linkage.h add MPQC_FORCELINK_KEYVAL_CTOR(T) to a .cpp file.

Note
Since for DescribedClass-based classes one needs to register the KeyVal ctor AND force their linking into the executable, usually this macro should not be used directly, instead use MPQC_CLASS_EXPORT2(Key,T) or MPQC_CLASS_EXPORT(T)
