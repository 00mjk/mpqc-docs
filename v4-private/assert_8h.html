MPQC_ASSERT(a, message) asserts that a is true ; if not, will produce a result that depends on the value of the MPQC_ASSERT_MODE macro

Note
message is optional
implementation follows the recipe from https://stackoverflow.com/questions/11761703/overloading-macro-on-number-of-arguments
