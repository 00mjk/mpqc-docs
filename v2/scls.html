The scls program is used to list objects in checkpoint files. See The State Library for more information.

Compiling scls

scls is distributed with the SC Toolkit. scls will automatically be compiled when SC is compiled. See Compiling for more information.

Running scls

scls takes the following command line options followed by a list of files.

-memorygrp val
Which memory group to use.
-threadgrp val
Which thread group to use.
-messagegrp val
Which message group to use.
-W val
Set the working directory.
-d
Turn on debugging.
-v
Print the version.
-w
Print the warranty.
-l
Detailed list of objects.
-L
Print the license.
-h
Print this help.
